# Syntax Package

This package provides syntax parsing/highlighting support for CotEditor.


## Tree-sitter Support

The following languages are currently supported by tree-sitter in this package:

- Bash
- C
- C#
- C++
- CSS
- Dart
- Dockerfile
- Go
- HTML
- Java
- JavaScript
- JSON
- Kotlin
- Lua
- Makefile
- Markdown (outline extraction only)
- PHP
- Python
- Ruby
- Rust
- Scala
- SQL
- Swift
- TypeScript
- XML
- YAML
